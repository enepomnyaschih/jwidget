    DOM element property management method.

    Returns a boolean property containing current checkbox state and starts watching for its modification. Destroy the result property to stop synchronization.

    // Watch checkbox state
var property = this.own(el.jwprop("checked"));

    • Defined in ui/prop/index.ts:61

    DOM element property management method.

    Binds specified property of the DOM element to boolean property and/or vice versa. Returns [[JW.UI.PropBinding]] instance. Destroy it to stop synchronization.

    // Bind element state to property
this.own(el.jwprop("disabled", property));

    Two way binding:

    this.own(el.jwprop("checked", this.value, JW.TWOWAY));

    Parameters

    • el: JQuery
    • prop: string

      Element's property name.

    • property: Property<boolean>

      Property value.

    • Optional binding: Binding

      Binding mode. Defaults to [[JW.Binding.UPDATE]].

    Returns IClass
