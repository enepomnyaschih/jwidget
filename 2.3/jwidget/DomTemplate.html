jwidget/DomTemplate

Consumption

import DomTemplate from "jwidget/DomTemplate";

Default export#

Hierarchy#

  • abstract class AbstractTemplate
  • class DomTemplate

Description#

Template that processes jwclass and jwid attributes in an existing DOM element. This kind of template is applied to a Component when an HTML element is passed to using method.

new MyComponent().using($("#sample")).render();

class MyComponent extends Component {
    protected renderMessage(el: JQuery) {
        el.text("Hello!");
    }
}
Input DOM structure
<div id="sample" jwclass="my-component">
    <div jwid="message"></div>
</div>
Output DOM structure
<div id="sample" class="my-component">
    <div class="my-component-message">Hello!</div>
</div>

Constructor#

new DomTemplate(el: HTMLElement | JQuery)

el
Element to process.

Fields#

requiresAfterAppend (inherited from AbstractTemplate)#

readonly requiresAfterAppend: boolean

Flag indicating if rendering of a Component by this template must be followed with a DOM presence check with a possible afterAppend method call.
