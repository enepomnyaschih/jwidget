jwidget/ui/attr

Consumption

import attr from jwidget/ui/attr;

Signature

(el: JQuery, attr: string, property: %Bindable): %Destroyable
el
DOM element.
attr
Attribute name.
property
Attribute value to assign.
Returns
Binding object. Destroy the returned object to stop synchronization.

Watches string property modification and updates the specified attribute of the DOM element.

// Bind "title" attribute to title property value this.%own(attr(el, "title", title));
